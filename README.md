## Projects Included

### 1. Active Directory + Group Policy (GPO)
Configured a domain environment with a secondary domain controller and applied password and lockout policies through Group Policy.

### 2. Exploiting Systems: Heartbleed (OpenSSL)
Simulated and exploited the Heartbleed vulnerability using a vulnerable OpenSSL Docker container and the Heartleech exploit tool.

### 3. Secure Linux Firewalls
Deployed two Ubuntu 20.04 VMs, configured users and sudo access, installed Apache and Nginx, enforced UFW firewall rules, and verified connectivity between systems.

### 4. Splunk Monitoring
Installed Splunk Enterprise on one Ubuntu VM and configured log collection and forwarding from a second Ubuntu VM using Splunk Universal Forwarder.
